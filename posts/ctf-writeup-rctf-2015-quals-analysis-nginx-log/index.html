Date Tue 17 November 2015 Tags ctf / sqli

Challenge

Writeup

For each character of the flag, there is a bunch of entries that basically does this:

if flag[0].char_code > 64:
    sleep(1)

Then, after multiple guesses, sqlmap confirms the exact character code:

if flag[0].char_code != 82:
    sleep(1)

Then it goes to the next character: LIMIT 0,1),1,1 becomes LIMIT 0,1),2,1.

So the most interesting entries look like this: ...LIMIT 0,1),1,1))!=82),SLEEP(1)... (highlighted in the snippet). We can find the flag value by extracting every character after a !=. I used Sublime Text's search + multi-select for this, but you can do some vim or regex magic too. Pasting these characters in my good old friend asciitohex revealed the flag. In the snippet above, you can find the first 4 characters: ROIS.
